BETTER ALTERNATIVE: DATA AUGMENTATION

  Instead of error data, use augmentation on normal data:
  
  • Node dropout: Randomly remove nodes
  • Edge dropout: Randomly remove edges
  • Feature noise: Add noise to attributes
  • Subgraph sampling: Extract subgraphs
  
  Benefits:
    ✓ Increases dataset size
    ✓ Improves robustness
    ✓ Prevents overfitting
    ✓ No manual work needed
    ✓ Already in PyTorch Geometric!
